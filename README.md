# Senpai Pathway
To get started: 
```
git clone git@github.com:JoshuaToth/SenpaiPathway.git
yarn install
npm run develop

Navigate to: http://Localhost:8000
```
To Publish:
```
npm run publish
```
This will publish to [Github Pages: https://joshuatoth.github.io/SenpaiPathway/ ](https://joshuatoth.github.io/SenpaiPathway/)
